1.配置 Git
# 设置用户名（重要，会影响提交记录）
git config --global user.name "你的名字"
# 设置邮箱
git config --global user.email "你的邮箱@example.com"
# 查看配置
git config --list

2.初始化本地仓库
git init

3.添加远程仓库地址
从 Gitee 复制仓库地址（HTTPS）
git remote add origin https://gitee.com/lzzzOdin/test-tushu.git
克隆项目（使用 SSH 地址）
git clone git@gitee.com:用户名/仓库名.git

查看是否添加成功
git remote -v
检查状态
git status

4.提交
git add *.py
git commit -m "首次提交：图书管理系统核心代码及文档"
git push -u origin master

5.拉取代码
git pull origin master
拉取并合并
git pull origin master --allow-unrelated-histories

python main.py
