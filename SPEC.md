# 图书数据管理系统 - 规格文档

## 1. 项目概述

- **项目名称**: 图舒图书管理系统
- **项目类型**: 桌面应用程序
- **核心功能**: 图书数据的增删改查、文件导入（ TXT/EPUB ）、阅读器
- **目标用户**: 个人图书阅读管理者

## 2. 技术栈

| 模块 | 技术选型 |
|------|----------|
| 界面框架 | Tkinter |
| EPUB解析 | ebooklib + BeautifulSoup4 |
| 数据存储 | JSON |
| Python版本 | Python 3.8+ |

## 3. 功能列表

### 3.1 图书数据管理模块
- **添加图书（手动录入）**: 输入书名、作者，自动生成图书ID，保存到本地数据库
- **删除图书**: 选中图书后删除，删除前二次确认
- **修改图书信息**: 修改书名、作者，保存修改结果
- **查询图书**: 按书名模糊搜索，实时筛选显示结果

### 3.2 文件导入模块
- **TXT文件导入**: 选择本地txt文件，解析文本内容，提取书名（文件名或首行），存储文件路径和内容
- **EPUB文件导入**: 选择本地epub文件，提取书籍元数据（书名、作者），解析章节内容，存储文件路径

### 3.3 阅读器模块
- **图书列表展示**: 显示所有图书（标题/作者），双击或点击图书进入阅读
- **阅读界面**: 显示文本内容（txt纯文本），显示EPUB章节内容，翻页/滚动阅读，返回图书列表

### 3.4 图形界面模块
- **主窗口**: 图书列表区域（中央），搜索框（顶部），操作按钮：添加、删除、修改、导入（顶部）
- **图书编辑对话框**: 书名输入框，作者输入框，确定/取消按钮
- **阅读窗口**: 文本显示区域，关闭/返回按钮

### 3.5 数据存储
- 使用JSON文件存储图书信息（ID、书名、作者、文件路径、文件类型）
- 原始图书文件保留在原位置

## 4. 数据模型

### 4.1 Book（图书）
```json
{
  "id": "string (UUID)",
  "title": "string (书名)",
  "author": "string (作者)",
  "file_path": "string (文件路径，可为空)",
  "file_type": "string (none/txt/epub)",
  "content": "string (文本内容，仅txt和手动录入时存储)",
  "created_at": "string (ISO格式时间)"
}
```

### 4.2 Storage（存储）
- 数据文件: `books.json`
- 存储路径: 应用运行目录

## 6. 模块架构

```
tushu/
├── main.py              # 程序入口
├── data_manager.py      # 数据管理模块（增删改查）
├── file_importer.py     # 文件导入模块（TXT/EPUB解析）
├── storage.py           # JSON数据存储
├── ui/
│   ├── __init__.py
│   ├── main_window.py   # 主窗口
│   ├── dialogs.py       # 对话框
│   └── reader_window.py # 阅读窗口
└── books.json           # 数据存储文件
```

## 7. 用户流程

```
启动程序 → 显示图书列表
    ├─ 搜索图书 → 输入关键词 → 实时显示筛选结果
    ├─ 添加图书 → 手动输入书名/作者 → 保存
    ├─ 导入文件 → 选择txt/epub → 自动解析元数据 → 保存
    ├─ 选中图书
    │   ├─ 修改信息 → 弹出编辑框 → 修改保存
    │   ├─ 删除 → 确认对话框 → 删除
    │   └─ 阅读 → 打开阅读窗口 → 显示内容 → 关闭返回
```

4. ✅ 可以修改图书信息
5. ✅ 可以按书名模糊搜索
6. ✅ 可以导入TXT文件并解析内容
7. ✅ 可以导入EPUB文件并解析元数据
8. ✅ 可以阅读TXT文本内容
9. ✅ 可以阅读EPUB章节内容
